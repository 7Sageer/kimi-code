Add plugin system for local-path skill installation and bootstrap injection. New `/plugins` slash command supports install, list, info, enable, disable, remove, and reload operations. Plugins contribute skill directories and optional bootstrap system reminders to new sessions. Includes compatibility shim for the Superpowers plugin to enable auto-triggered brainstorming workflows.
